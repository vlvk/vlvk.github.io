React Hooks 简单体验

2024-10-31[最近更新::2024-11-15]VLVK2 min read (351 words)
#Frontend  #React 

1. 简介#

简而言之：让函数组件拥有类组件的能力。

import React from 'react';
class MyCount extends React.Component {
    state = {
        count: 0
    }
    componentDidmount() {
        this.interval = setInterval(() => {
            this.setState(, 1000);
        });
    }
    componentWillUnmount() {
        if ( this.interval ) {
            clearInterval(this.interval);
        }
    }
    render() {
        return (
            <span>{this.state.count}</span>
        )
    }
}

由于无法函数组件没有 this，在 React Hooks 出来之前无法实现

function MyCountFunc() {
    const [ count, setCount ] = useState(0);  // Hook

    useEffect(() => {
        const interval = setInterval(()=>{
            setCount(x => x + 1);
        }, 1000);
        return () => clearInterval(interval);  // callback
    }, []);
}

2. State Hook#

两种API:

  • useState
  • useReducer

useState 是基于 useReducer 实现，useReducer 才是基础的 State API.

  1. useState

useState 返回值为一个数组 [a, b]

const [ count, setCount ] = useState(0); // 默认值只在初次渲染时有效，后续更新渲染初始值不起作用。

两种参数传入方式：

  • 直接赋值, setCount(0)
  • 回调函数（避免于陷于闭包陷阱）
setCount(x => x + 1)
  1. useReducer

类似于 Redux, 创建一个 Reducer 函数，接收 action .

function MyCountReducer (state, action) {
    switch ( action.type ) {
        case 'add': return {
            ...state,
            count: count + 1;
        }
        case 'minus': return {
            ...state,
            count: count - 1;
        }
        default : return state;
    }
}

使用：

const [ count, dispatchCount ] = useReducer(MyCountReducer, 0);
...
const interval = setInterval(()=>{
    // setCount(x => x + 1);
    dispatchCount({ type: 'add' });
}, 1000);
...

3. Effect Hook#

  • useEffect
  • useLayoutEffect，会在组件挂载之前运行。
useEffect(() => {
    ...
}, []);

当数组中元素变换时，重新运行 useEffect 函数，数组中的元素被称为依赖。

4. Context Hook#

  • use Context
// my-context.js
import React from 'react';
export default React.createContext('');

随后的使用过程中

...
import MyContext from '../lib/my-context';
import Component from './b.js';
...
render() {
    return (
        <MyContext.Provider value="test">
            <Component />
        </MyContext>
    );
}

// b.js
import { useContext } from 'react';
import MyContext from '../lib/my-context';

const context = useContext(MyContext);
return (
    <div>{ context }</div>
);

5. Ref Hook#

传统类组件可以通过添加 Ref 字段来获取 DOM 对象，string ref 17版本删掉

this.ref = React.createRef()
...
componentDidMount() {
    console.log(this.ref.current) // 获取DOM对象
}

return (
    <span ref={this.ref}>
);

import { useRef } from 'react';

const inputRef = useRef();

useEffect(() => {
    console.log( inputRef.current );
}, []);

return (
    <input ref={ inputRef } />
);

6. 渲染优化#

必要优化点

  1. memo 提供函数组件类似于类组件 shouldComponentUpdate 对比变化函数。取决于函数参数是否存在变化
memo(
    function component
);
  1. useMemo
useMemo(() => {}, [依赖注入])
  1. useCallback 简化使用版本的 useMemo
useCallback(() => {}, [依赖注入]);
